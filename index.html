What is Kjell?

Kjell (IPA: [/ʃɛl/]) is an refurbishing of the Erlang Shell, with some eye-candy and plugin support added. To build Kjell, execute the following commands in a terminal window :


$ git clone https://github.com/karlll/kjell.git
$ cd kjell
$ make deps
$ make

Start Kjell : 

bin/kjell

Dependencies

To build Kjell the following is needed

  • Autoconf
  • Curses
  • gcc
